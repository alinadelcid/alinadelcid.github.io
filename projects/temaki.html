Temaki design system: UI specifics for a streaming website

Temaki is a streaming service that aims to create an exciting, theatre-like experience for a wide audience. I was tasked with developing the Temaki Design System, a comprehensive set of guidelines for creating web-based visuals. This design system was created as a detailed, one-page interactive prototype featuring a vertical navigation bar. Each option in the nav bar scrolls to the corresponding section of the visual guidelines, ensuring easy navigation and accessibility.
