OVERVIEW

Outlining UI specifics for a streaming website

Temaki is a streaming service that aims to create an exciting, theatre-like experience for a wide audience. The Temaki Design System is Temaki’s guidelines for creating web-based visuals. I was tasked to create a detailed web-based design system that would be implemented into the web design.

DELIVERABLES

Design System Website

The design system was developed as a one-page interactive prototype with a vertical nav bar. Each option in the nav bar scrolls to the corresponding section of the visual guidelines.
